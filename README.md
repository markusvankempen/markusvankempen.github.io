# markusvankempen.github.io

Personal developer site for **Markus van Kempen** — showcasing open-source extensions and tools for IBM Maximo API development.

**Live site:** [https://markusvankempen.github.io](https://markusvankempen.github.io)

## What's Here

A single-page site built with IBM Carbon Design System styling that highlights:

| Tool | Platform | Description | Links |
|------|----------|-------------|-------|
| **Maximo API Explorer** | VS Code | Discover, browse, and test IBM Maximo REST APIs directly from VS Code | [Marketplace](https://marketplace.visualstudio.com/items?itemName=MarkusvanKempen.maximo-api-explorer) • [Open VSX](https://open-vsx.org/extension/markusvankempen/maximo-api-explorer) • [Source](https://github.com/markusvankempen/maximo-mcp-ai-integration-options/tree/main/maximo-api-explorer) |
| **Maximo Cursor Explorer** | Cursor | AI-optimized API explorer with `.cursorrules` and context export | [Marketplace](https://marketplace.visualstudio.com/items?itemName=MarkusvanKempen.maximo-cursor-explorer) • [Open VSX](https://open-vsx.org/extension/markusvankempen/maximo-cursor-explorer) • [Source](https://github.com/markusvankempen/maximo-mcp-ai-integration-options/tree/main/maximo-cursor-extension) |
| **Maximo MCP Server** | Claude / ChatGPT / Copilot | Model Context Protocol server for AI-assisted Maximo development | [Marketplace](https://marketplace.visualstudio.com/items?itemName=MarkusvanKempen.maximo-mcp) • [Open VSX](https://open-vsx.org/extension/markusvankempen/maximo-mcp) • [npm](https://www.npmjs.com/package/maximo-mcp-server) • [Cursor Directory](https://cursor.directory/mcp/maximo-mcp-server) • [Source](https://github.com/markusvankempen/maximo-mcp-ai-integration-options) |

## Tech Stack

- Pure HTML + CSS (no build tools)
- IBM Plex Sans font (Google Fonts)
- Carbon Design System color tokens & spacing
- Responsive (mobile-friendly)

## Deployment

Hosted via GitHub Pages. Push to `main` and the site updates automatically.

```bash
git add -A
git commit -m "Update site"
git push origin main
```
